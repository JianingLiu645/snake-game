# 贪吃蛇小游戏项目

两个纯 HTML 的贪吃蛇游戏（无需构建，直接在浏览器打开即可运行），中文界面。

## 文件说明

| 文件 | 说明 |
|---|---|
| `snake.html` | **基础版** — 经典贪吃蛇，带分数、最高分、粒子特效 |
| `snake-pro.html` | **高级版** — 多种食物、能力道具、生命系统、关卡升级、暂停功能 |

## 基础版特性 (snake.html)

- 400×400 Canvas，20×20 网格
- 键盘控制（方向键 + WASD）+ 触屏方向键 + 滑动手势
- 吃食物得 10 分，最高分存 localStorage (`snakeBest`)
- 粒子爆炸特效
- 蛇头有眼睛动画（随方向转动）
- 游戏速度：120ms/帧
- 深色主题 (#1a1a2e)

## 高级版特性 (snake-pro.html)

- 三种食物：普通（红色 +10）、金币（黄色 +30）、毒药（紫色 -15）
- 三种能力道具：⚡加速、🛡护盾（挡一次碰撞）、×2双倍分
- 3 条命，撞墙/撞自身扣一条命，保留分数继续
- 关卡系统：每吃 5 个食物升级，速度递增（130ms → 55ms）
- 连击计数
- 暂停/继续（空格键或 Esc）
- 开始界面、暂停遮罩、游戏结束遮罩
- Apple 风格 UI：CSS 变量、毛玻璃效果、圆角卡片
- 浮动文字特效、粒子带重力
- 响应式布局，移动端触屏方向键

## 技术栈

- 纯 HTML/CSS/JS，零依赖
- Canvas 2D 渲染
- localStorage 持久化最高分
- 直接在浏览器打开文件即可运行

## 运行方式

```bash
# 任意方式打开即可
start snake.html        # Windows
start snake-pro.html    # Windows
open snake.html         # macOS
```
